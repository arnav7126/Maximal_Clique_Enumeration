Tomita's Algorithm

Tomita's Algorithm is an optimized algorithm for maximal clique enumeration using branch-and-bound pruning.

  • Operates in three main sets:
    • R (current clique): Vertices that form a clique.
    • P (potential candidates): Vertices that can still be added to R.
    • X (already explored vertices): Vertices that were part of previous cliques.
  • Uses a pivot selection strategy to minimize recursive calls and improve efficiency.
  • Prunes the search space by skipping branches that cannot form larger cliques.
  • Improves on the Bron-Kerbosch algorithm by reducing redundant searches.
